Part 2: Bootstrap Historical Simulation method¶

Using the Bootstrap method, we compute for each new sample the 10 b-days return formula : $\prod_{i=0}^9 (R_{i,i+1} +1) - 1$

In [67]:
#Bootstrap des donnes

k = 10000 #Sample size
d = 10 #Time frame

sample_ddays_return = []
for _ in range(k):
    
    #We prepare a sample of 10 daily returns (w replacement) and add 1 to each coordonate
    sample_n = np.random.choice(Ptf_Return, size=d) + [1]*d
    
    #Computation of the 10-days return for this sample
    sample_ddays_return.append(np.prod(sample_n)-1)

Now we've got a vector of 10000 10-business days returns, let's compute the 10 business days VaR(99).

In [68]:
